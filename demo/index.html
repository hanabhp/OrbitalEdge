OrbitalEdge: Cross-layer NTN Handover Prediction

LEO satellites at 530 km orbit create handovers every 8-18 s for a moving UE. Existing 3GPP Conditional Handover fires only 2 s ahead because RSRP/SINR at the D2C floor are statistically indistinguishable from normal fading.

OrbitalEdge reads protocol-layer features (DL/UL TBS ratio, MR density, PDCCH aggregation level) that encode the handover signature 8-12 s in advance, then sends CHO preparation to the gNB via the O-RAN Near-RT RIC A1 interface.

Amber satellites are active in the constellation. Gold = serving UE. Teal = predicted next-serving. The progress bar at top shows time to the next handover; the red marker shows where OrbitalEdge fires. The sidebar plots the four key protocol invariants in real time.
